✅ Checklist — App SIBT

Siga os passos abaixo para publicar o app da Segunda Igreja Batista do Tabuleiro

0 de 15 passos concluídos
⚠️ Antes de começar — leia isso
Você vai precisar abrir alguns sites. Em cada passo, há um link azul que abre o site certo. Marque o passo como ✓ quando terminar cada um. Pode pausar e continuar depois — o progresso fica salvo.
Parte 1 — Configurar o Firebase (banco de dados)
1
Criar conta no Firebase com o Gmail da SIBT
Entre com o Gmail igrejasibt@gmail.com e crie um projeto chamado SIBT App.
Abrir firebase.google.com ↗
✓
2
Ativar o login pelo Gmail
No projeto → Authentication → Provedores de login → Google → Ativar → E-mail de suporte: igrejasibt@gmail.com → Salvar.
✓
3
Criar o banco de dados (Firestore)
No projeto → Firestore Database → Criar banco de dados → Modo produção → Região: southamerica-east1 → Ativar.
✓
4
Criar as coleções no banco de dados
Clique em "+ Iniciar coleção" e crie uma por vez, com um documento vazio (botão Auto-ID) em cada:

members · events · finance · balancetes · atas · turmas · materiais · pregacoes · inscricoes

São 9 coleções no total. Após criar, pode apagar o documento vazio de exemplo.
✓
5
Configurar as regras de segurança
Firestore → aba "Regras" → apague tudo e cole o conteúdo do arquivo firestore.rules que está na pasta sibt-app. Clique em Publicar.
✓
6
Copiar as credenciais do Firebase
Engrenagem → Configurações do projeto → role até "Seus aplicativos" → clique em "</>" → apelido: sibt-web → Registrar. Copie os valores do bloco firebaseConfig e guarde num bloco de notas.
✓
Parte 2 — Editar o arquivo do app
7
Abrir a pasta sibt-app no VS Code
Instale o VS Code se ainda não tiver. Abra o programa → File → Open Folder → selecione a pasta sibt-app que você recebeu.
Baixar VS Code ↗
✓
8
Colar as credenciais no arquivo index.html
No VS Code, abra o arquivo index.html. Use Ctrl+F para buscar: COLE_AQUI_SUA_apiKey

Substitua cada valor COLE_AQUI_... pelo valor real que você copiou no passo anterior. São 6 valores para substituir.
apiKey: "AIzaSy..." ← cole o valor real authDomain: "sibt-app.firebaseapp.com" projectId: "sibt-app" storageBucket: "sibt-app.appspot.com" messagingSenderId: "12345..." appId: "1:12345..."
✓
Parte 3 — Publicar no GitHub
9
Criar conta no GitHub
Crie com o Gmail da SIBT. Nome de usuário sugerido: sibtabuleiro. Confirme o e-mail.
Criar conta no GitHub ↗
✓
10
Criar repositório no GitHub
Clique em "New repository" → Nome: sibt-app → Public → Create repository. Copie o endereço (ex: github.com/sibtabuleiro/sibt-app).
✓
11
Instalar o Node.js
Baixe e instale a versão LTS. Após instalar, abra o Prompt de Comando e confirme digitando node --version.
Baixar Node.js ↗
✓
12
Subir o código para o GitHub
No VS Code, abra o Terminal (menu Terminal → New Terminal) e execute os comandos abaixo um por vez:
git init git add . git commit -m "App SIBT com Firebase" git branch -M main git remote add origin https://github.com/sibtabuleiro/sibt-app.git git push -u origin main
✓
Parte 4 — Publicar online no Vercel
13
Publicar no Vercel (3 cliques)
Entre com o GitHub → Add New Project → selecione sibt-app → Import → Deploy. Aguarde 1 minuto. O app estará em: sibt-app.vercel.app
Abrir Vercel ↗
✓
14
Autorizar o domínio no Firebase
Firebase → Authentication → Settings → Domínios autorizados → Adicionar domínio → cole: sibt-app.vercel.app → Adicionar. Sem isso o login pelo Gmail não funciona.
✓
Parte 5 — Definir você como administrador
15
Entrar no app e se tornar admin
1. Abra o app: sibt-app.vercel.app
2. Toque em "Entrar" → "Entrar com Google" → selecione o Gmail
3. Volte ao Firebase → Firestore → coleção members
4. Encontre seu documento → clique em Editar → adicione o campo:
  perm (tipo: string) = admin
5. Salve → reabra o app → você é administrador! 🎉
✓
🎉 Pronto! App publicado.

Android: Chrome → três pontos → "Adicionar à tela inicial"
iPhone: Safari → botão compartilhar → "Adicionar à Tela de Início"
